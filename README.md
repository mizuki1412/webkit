## 环境变量

- NODE_ENV: production
- BASE_URL in vue.config.js
- ENTRY in vue.config.js
- VUE_APP_ROUTE_BASE
- VUE_APP_ROUTE_MODE: hash or history
- VUE_APP_BASE_URL: 后端请求的base url
- VUE_APP_TEST_BASE_URL: dev环境下的后端请求的base url
